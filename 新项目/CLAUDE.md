# CLAUDE.md

## 项目概况
萌宠消消乐 — 休闲消除类 H5 小游戏，纯前端实现，可直接在浏览器中运行。

## 项目结构
- `index.html` — 游戏界面，包含棋盘、分数面板、按钮
- `style.css` — 深色主题样式、动画（消除/下落/交换/连击）
- `script.js` — 完整 Match-3 游戏逻辑

## 技术栈
- 纯 HTML/CSS/JavaScript，无框架无依赖
- 8×8 棋盘，6 种动物 emoji
- CSS 动画驱动的消除/交换/下落效果
- 触摸和点击双支持

## 核心机制
- 点击选中 → 点击相邻 → 交换 → 检测 3+ 匹配 → 消除 → 重力下落 → 填充 → 联级检测
- 连击倍率：连锁消除时 combo 递增，分数 = 消除数 × 10 × combo
- 30 步限制，步数用尽游戏结束
- 智能提示：高亮显示一对可交换的棋子
- 无解自动洗牌
